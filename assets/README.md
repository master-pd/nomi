# Assets Directory

This directory contains all static assets for the NOMI bot.

## Structure

assets/
├── fonts/           # Font files for image generation
│   ├── kalpurush.ttf
│   └── siyamrupali.ttf
├── templates/       # Image templates
│   ├── welcome_template.png
│   └── goodbye_template.png
└── bg.jpg          # Default background image

## Usage

- Fonts are used for text rendering in images
- Templates are used for welcome/goodbye images
- Background images are used for various image generations

## Adding New Assets

1. Place font files in fonts/
2. Place image templates in templates/
3. Update config files to reference new assets
